Icon

RAD CSS is built with icons at its core, utilizing Google Icons to offer features like customizable weight and fill, allowing you to perfectly match your website's design.

Syntax

Icons can be added to <button>, <a>, <header> and more by simply adding the data-icon="icon_name" attribute. "icon_name" can be any icon name found in the Goole Icons Library.

Place the icon on the top, left, right, or bottom using the data-placement attribute.

Label
<a data-icon="arrow_forward" data-placement="right">Label</a>

Alternate Syntax

Icons can also be added to a <div> with the .data-icon class. This method is for situations where data-icon is not viable, like a button with a tooltip or a icon button with no innerHTML. 

                        
<button data-tooltip="next">
    <div class="data-icon">arrow_forward</div>
</button>

Customization

As illustrated above, RAD CSS uses three main variable to control the look and feel of icons: padding[Black], border-width[Blue], and font-size [White].

Additionally, the Google Icon attributes can be modified according to the Material Design Icon Guidelines.

Info

Buttons are designed such that the icon perfectly fits, regardless of the variable values.
