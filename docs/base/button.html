Button

Buttons are defined symantically and include the ability to add icons using the data-icon attribute.

Syntax

<button>Label</button>

Utility Classes

Styles

Buttons come with multiple utility classes for different commonly used styles and use-cases.

Utility
.text
.text.border
.large

Color

Change the color using color utility classes. Learn more about colors here.

Utility
.bg-tertiary
.bg-onerror
.bg-inversesurface

Icons

Easily add icons to buttons with data-icon. Learn more about icons here.

Syntax
<button data-icon="home">Label</button>
<button data-icon="home" data-placement="right">Label</button>
<button data-icon="home" data-placement="top">Label</button>
<button data-icon="home" data-placement="top"> Label </button>
<button> <div class="data-icon">Home</div> </button>

Multiple Icons

Due to the way data-icon is implemented, multiple icons are achieved with a nested <div class="data-icon">. 

                        
<button>
    <div class="data-icon">Home</div>
    Home
    <div class="data-icon">Home</div>
</button>

Add .vertical to achieve a vertical stacking with multiple icons. 

                        
<button class="vertical">
    <div class="data-icon">Home</div>
    Home
    <div class="data-icon">Home</div>
</button>
