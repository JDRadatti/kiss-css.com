Spacing

RAD CSS utilizes the golden ratio for satisfying spacing.

Overview

Padding and margins are defined with the spacing variables:

  • --xxs
  • --xs
  • --sm
  • --md
  • --lg
  • --xl
  • --xxl

These variables ensure satisfying spacing and are easily customizable for a unique look and feel.

Spacing utility classes are in the form [p/m]-[top/right/bottom/left]__[size] where "p" stands for padding and "m" stands for margin. For example, use the following classes to add padding to the top of an element:

  • p-top__xxs
  • p-top__xs
  • p-top__sm
  • p-top__md
  • p-top__lg
  • p-top__xl
  • p-top__xxl

Dynamic Spacing

Spacing dynamically adapts to the current font size. For example, var(--lg) is larger for .display2 than .title3.

Font Size Default Element
.display1 <h1>
.display2 <h2>
.title1 <h3>
.title2 <h4>
.title3 <h5>
.heading <h6>
.subheading N/A
.body <p>

Customization

Spacing can be linearly scaled with Sass using the $scaleFactor variable.

The spacing ratio can be changed with Sass using the $scaleRatio variable. This variable defaults to the golden ratio, 1.618.
