Navigation

Navigation uses the <nav> tag and adapts to vertical and horizontal positionings.

Syntax

Note

By design, the navigation bar does not have padding or margin. The spacing is handled by layout.

  • Brand
  • Home
  • Next
                    
<nav>
    <ul>
        <li class="title3__bold"><a>Brand</a></li>
    </ul>
    <ul>
        <li><a>Home</a></li>
        <li><a>Next</a></li>
    </ul>
</nav>

Aside

Navigation changes to a vertical layout when in a <aside> element or if role="menu".

  • Brand
  • Home
  • Next
                    
<nav role="menu">
    <ul>
        <li class="title3__bold"><a>Brand</a></li>
    </ul>
    <ul>
        <li><a>Home</a></li>
        <li><a>Next</a></li>
    </ul>
</nav>

        <ul>
            <li class="title3__bold"><a>Brand</a></li>
        </ul>

                        <summary>Summary 1</summary>
                        <ul>
                            <li><a>Link 1</a></li>
                            <li><a>Link 2</a></li>
                        </ul>
                    </details>
                </li>
            </ul>
        </details>
        <details>
            <summary>Summary 2</summary>
            <ul>
                <li><a>Link 3</a></li>
                <li><a>Link 4</a></li>
            </ul>
        </details>
    </nav>
</aside>

Disabled

Disable the dropdown effect by adding disabled to the <summary> tag.

Disabling the dropdown hides the arrow.

Summary
  • Link 1
  • Link 2
                    
<aside>
    <nav>
        <details open>
            <summary disabled>Summary</summary>
            <ul>
                <li><a>Link 1</a></li>
                <li><a>Link 2</a></li>
            </ul>
        </details>
    </nav>
</aside>

Hide Arrows

Use the .hide-arrows class on the <summary> tag.

Spacing

Navbar uses diplay: flex with flex-direction: space-between to evenly space <ul> elements.

  • Brand
  • Link
                    
<nav>
    <ul>
        <li class="title3__bold"><a>Brand</a></li>
    </ul>
    <ul>
        <li><a>Link</a></li>
    </ul>
    <ul>
        <li><button>Button</button></li>
    </ul>
</nav>

Horizontal Rule

Add a <hr /> to the navbar to get a vertical separator between links.

  • Home
  • About
                    
<nav>
    <ul>
        <li><a>Home</a></li>
        <hr />
        <li><a>About</a></li>
    </ul>
</nav>
