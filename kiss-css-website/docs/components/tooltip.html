Tooltip

Add a tooltip to any element with the data-tooltip attribute.

Syntax

<button data-tooltip="Button">Button</button>
Link
<a data-tooltip="Link">Link</a>

Placement

Change the placement with the data-placement attribute.

Syntax
<button data-tooltip="Right" data-placement="right">Label</button>
<button data-tooltip="Left" data-placement="left">Label</button>
<button data-tooltip="Top" data-placement="top">Label</button>
<button data-tooltip="Bottom" data-placement="bottom">Label</button>
