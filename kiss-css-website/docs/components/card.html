Card

Cards are semantically defined using the <article> tag. By default, their padding and border-radius vary based on the largest font size inside the card.

Dynamic Sizing

Card padding and border radii automatically scale to match the size of the largest font size within the card.

Static Sizing

It is possible to turn off the dynamic padding and border radius of cards.

Simply set $dynamic-card-padding: false; in /sass/abstracts/_config.scss.
