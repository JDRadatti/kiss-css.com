Dropdown

Use <details> with role="dropdown" attribute to create a dropdown.

Syntax

Graduation Requirements
  • Requires 40 credits
  • Including a passing grade in health
  • Geography
  • History
  • Economics
  • Wood Shop
                        
<details role="dropdown">
    <summary>Graduation Requirements</summary>
    <ul>
        <li>Requires 40 credits</li>
        <li>Including a passing grade in health</li>
        <li>Geography</li>
        <li>History</li>
        <li>Economics</li>
        <li>Wood Shop</li>
    </ul>
</details>
