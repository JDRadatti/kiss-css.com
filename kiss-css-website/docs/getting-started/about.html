About

KISS CSS is designed with simplicity and customization at the forefront.

Goals

  • Create a drag-and-drop css file to instantly make any website beautiful.
  • Prevent cookie cutter designs. KISS CSS makes it easy to design completely distinct website.
  • Powerful builtin features like automatic layouts.
  • Simple and satisfying spacing.
  • Only CSS. Just add KISS CSS to your symantic HTML and you're done.

Built with

This website is primarily built with KISS CSS. The only additonal CSS added to this website is for the code highlighting (Prism) and a couple of flex boxes (builtin responsive grids and flexboxes coming soon).

This website also uses additional JavaScript, but mainly web components for reusing elements like the navbar, menu, sidebar, and footer.
