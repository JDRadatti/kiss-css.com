Customization

KISS CSS uses Material Design Theme Builder. Visit the Material design website to create and download your custom color scheme. Then, follow the instructions below to add the new colors to your website.

Classes

Overload any color with the classes downloaded from Material Design. Note the prefix --md-sys-color can be customized with Sass.

                    
<style>
    .theme-name {
        // Default colors
        --md-sys-color-primary: ...;
    }

    [data-theme="light"] {
        .theme-name {
            // Light colors
            --md-sys-color-primary: ...;
        }
    }

    [data-theme="dark"] {
        .theme-name {
            // Dark colors
            --md-sys-color-primary: ...;
        }
    }
</style>
</div class="theme-name"></div>

Sass

  • Clone the Repository.
  • Modify sass/theme/_colors.scss with the downloaded colors for both light and dark mode.
  • Recompile CSS file. Refer to the Sass documentation for more information.
Warning
KISS CSS uses three extended colors with Material Design Theme Builder: green, yellow/orange, and blue. green is used for success, yellow or orange is used for warnings(like this card), and blue is used for neutral information.
To achieve this:
  • Add the three extended colors when creating your color scheme.
  • Rename each extended color to their respective symantic meaning (i.e. rename the green extended color to "Success").
  • When importing into your stylesheet, remove the "--md-extended-color" prefix and replace with the "--md-sys-color" prefix, or #{$color-prefix} if using Sass.

Light and Dark Color Schemes

KISS CSS automatically user's system presets for color scheme. To force a specific color scheme, simply add data-theme="light" or data-theme="light" in the <html> tag.
