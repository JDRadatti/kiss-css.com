Link

Links have four styles, each with an aria-current style to signify the current page.

Syntax

Label
<a>Label</a>

Styles

Utility
  • Primary
  • Primary Current
  • Primary
 .primary
  • Secondary
  • Secondary Current
  • Secondary
 .secondary
  • Tertiary
  • Tertiary Current
  • Tertiary
 .tertiary
  • Contrast
  • Contrast Current
  • Contrast
 .contrast

Current Page

Use aria-current attribute to set style to current page.

<a aria-current="page">Label</a>
