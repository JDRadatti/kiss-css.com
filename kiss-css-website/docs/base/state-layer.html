State-layer

Overview

State layers power the hover and active states of KISS CSS. By default, <button> and <li> in dropdown navbars are the only elements that uses state layers. However, state layers can easily be added to other elements, such as <article>, just add .state-layer.

State layers are defined as a overlayed surface in ::after. This is designed to change properties not otherwise able to be modified, like opacity or animation.

Color
