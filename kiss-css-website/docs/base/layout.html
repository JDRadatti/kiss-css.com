Layout

Layout is powerful and simple. It supports a navbar, footer, menu, and a sidebar automatically, no setup needed. Layout is responsive and looks great on any screen size.

Overview

Define a layout symantically as seen below. The first <aside> is the menu (on the left), and the second <aside> is the sidebar (on the right).

If there is only one <aside>, it is treated as the menu.

If no <aside> is present, there will be no menu or sidebar (see the homepage as an example).

By default, <header> (the navbar) is sticky, but can easily be updated.

                    
<body>

    <!-- Navbar -->
    <header class="wide"> <kiss-navbar></kiss-navbar> </header>

    <!-- Main -->
    <main>

        <!-- Page Header -->
        <hgroup></hgroup>

        <!-- Section -->
        <section></section>

    </main>

    <!-- Menu -->
    <aside></aside>

    <!-- Sidebar -->
    <aside></aside>

    <!-- Footer -->
    <footer></footer>

</body>
